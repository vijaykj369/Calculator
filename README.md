# 🧮 Calculator Application
A simple and user-friendly Calculator Application built completely using **Java Swing** and **AWT**. This desktop application performs basic arithmetic operations with an intuitive graphical user interface.

## 🚀 Features

- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Decimal number support
- Clear (C) functionality
- Delete/Backspace functionality
- User-friendly GUI
- Error handling for invalid operations

## 🛠️ Technologies Used

- Java
- Swing (GUI Components)
- AWT (Event Handling & Layout Management)

## ⚙️ Requirements

- Java JDK 8 or higher
- Any Java IDE (IntelliJ IDEA, Eclipse, NetBeans, VS Code)

🧠 Working Principle

The application uses:

Swing Components such as:
JFrame
JPanel
JTextField
JButton
AWT Event Handling:
ActionListener
ActionEvent

When a button is clicked, an event is triggered and the corresponding arithmetic operation is performed. The result is displayed in the calculator screen.
